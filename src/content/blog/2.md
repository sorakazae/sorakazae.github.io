## 🎵 왜 Dynamic Lyrics를 찾게 되었는가

Spotify를 메인 스트리밍 서비스로 쓰다 보면,  
**음악 조작은 편한데 가사는 보기 불편한 상황**이 자주 생긴다.

나는 평소에 Mac에서 음악을 조작할 때  
`Spotify 앱을 직접 켜지 않고 Raycast로 모든 제어를 하는 방식`을 선호한다.

- 재생  
- 일시정지  
- 다음곡 / 이전곡  
- 플레이리스트 이동  

이 모든 것을 Raycast로 처리할 수 있다.  
문제는 하나였다.

> **가사는 Spotify 앱을 직접 열지 않으면 볼 수 없다.**

음악은 백그라운드로 재생하지만,  
가사를 보려면 굳이 Spotify UI를 띄워야 하는 건 매우 불편했다.

그래서 “가사를 PIP처럼 화면에 고정해주는 도구 없나?” 하고 검색을 시작했고,  
그때 발견한 게 바로 **Dynamic Lyrics**였다.

---

## 🪄 Dynamic Lyrics — 내가 원하던 기능이 이미 존재했다

처음 설치하자마자 무료 체험이 제공되는데,  
Spotify 연동 과정을 요구하지 않고도 **바로 현재 재생 중인 곡의 가사를 화면 상단에 띄워주는 기능**을 볼 수 있었다.

PIP처럼 고정되는 오버레이 형태로  
- 위치 조절  
- 가사 크기  
- 투명도  

이런 것들을 바로 설정할 수 있다.

기능을 보고 바로 이거다 싶어서  
그 자리에서 **평생 이용권(14,000원)을 구매함.**

그리고 놀라운 건  
> **아이폰에서도 동일한 라이선스를 그대로 사용할 수 있었다.**

아이폰에 설치해보니  
똑같이 라이선스가 연동되었고,  

Dynamic Island처럼  
**화면 상단에 실시간으로 가사가 노출**되는 모드를 지원하고 있었다.  
(이건 설정을 몇 가지 켜줘야 활성화됨)

Spotify / Apple Music 어디를 쓰든 동일하게 작동하는 점도 좋았다.

## 🌐 번역 기능이 생각보다 훨씬 강력하다

이 앱을 더 쓰다 보니까 진짜 인상 깊은 기능이 있었는데 바로 **번역 기능**이다.

Dynamic Lyrics는  
- 원본 가사  
- 번역된 가사  

두 가지를 동시에 띄울 수 있다.

그리고 번역 품질은  
기대 이하가 아니라 **예상 이상**이었다.

내 경우 J-POP을 자주 듣다 보니,  
일본어 가사 + 한국어 번역을 켜놓고 쓰는데  
앱 내부에서 언어 데이터베이스를 내려받기만 하면 된다.

> 오버레이 가사 + 원본/번역 동시 표시  
이 조합은 단순한 가사 앱 수준을 넘는다.

---

## 📝 가사가 없을 때는 직접 .lrc 파일을 만들어 연결할 수 있음

문제는,  
라이브 버전이나 인디곡처럼  
가사 DB에 등록되지 않은 곡들도 존재한다는 점이다.

이럴 때 Dynamic Lyrics는  
직접 `.lrc` 파일을 연결할 수 있는 기능을 제공한다.

파일의 형식은 단순하다:
```
[00:05.00] 첫 번째 가사
[00:10.00] 두 번째 가사
[00:16.50] 세 번째 가사
```
시간과 가사를 적기만 하면 되고,  
파일만 맞춰 두면 앱에서 바로 동기화된다.

다른 앱들은 이런 기능을 제한하거나 제공하지 않는 경우가 많은데,  
이 앱은 기본 기능처럼 자연스럽게 붙어 있다.

## 📚 PetitLyrics, Musixmatch 등 다른 DB에 직접 가사 등록도 가능

추가로,  
Dynamic Lyrics는 Spotify/Apple Music에서 불러오는 가사를 그대로 표시하는 앱이기 때문에,
스트리밍 서비스가 사용하는 가사 데이터베이스(PetitLyrics, Musixmatch 등)에 내가 직접 가사를 등록하면
Dynamic Lyrics에도 동일하게 적용된다.

일본에서 많이 사용하는  
**プチリリ(PetitLyrics)**  
그리고 글로벌 DB인  
**Musixmatch**  

같은 플랫폼에  
내가 직접 가사를 등록해두면  
Spotify / Apple Music에서 가사 싱크가 잡히고,  
Dynamic Lyrics에서도 그대로 적용된다.

이건 꽤 강력하다.  
내가 좋아하는 곡에 가사가 없다면  
직접 등록해서 생태계를 보완할 수 있기 때문이다.

---

## 🧩 실제 사용하면서 느낀 장점 요약

- Spotify 앱을 열 필요가 없다  
- Raycast + Dynamic Lyrics 조합이 완벽  
- Mac / iPhone 동일 라이선스  
- Dynamic Island 스타일 가사 가능  
- 번역 기능이 놀라울 정도로 자연스러움  
- 없는 가사는 .lrc로 직접 연결 가능  
- PetitLyrics / Musixmatch DB와 연동  

정말 **“음악을 조작은 Raycast로, 가사는 Dynamic Lyrics로”**가  
최적의 조합이라고 느꼈다.
